# 记忆秘诀

记忆秘诀： 括点油调一成做衣 叫等位裸跳服展昭

解释:

  • 括： 括号
  • 点： 成员访问
  • 油：new
  • 调：函数调用
  • 一：一元运算符
  • 成：乘除加减
  • 做衣： 按位左右移
  • 叫：比较运算符（大于 小于）
  • 等：等于
  • 位：按位与或
  • 裸：逻辑与或
  • 跳：条件运算符
  • 服：赋值
  • 展：展开运算符
  • 昭：逗号
